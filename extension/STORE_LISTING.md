## Permission Justifications

**Chrome Web Store requires explanations for each permission:**

**storage:**
```
Store user settings including API endpoint, API key, enabled platforms, and protection mode preferences. This allows your configuration to persist across browser sessions.
```

**activeTab:**
```
Detect when you're on a supported AI chat platform (ChatGPT, Claude, Gemini, GitHub Copilot) to activate InferShield's protection. We only access the active tab when you're on a supported site.
```

**host_permissions (chat.openai.com, claude.ai, gemini.google.com, github.com):**
```
Inject our content script into supported AI chat platforms to scan messages before you send them. InferShield only works on the platforms you explicitly enable in settings. We do not access any other websites.
```
