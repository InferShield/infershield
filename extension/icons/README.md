# InferShield Browser Extension Icons

## Required Icons

You need to create 3 PNG icons with the following sizes:

### 1. icon16.png (16x16 pixels)
- Small icon for browser toolbar
- Use a shield emoji or simple shield shape
- Colors: #00ff88 (green) on dark background

### 2. icon48.png (48x48 pixels)
- Medium icon for extension management page
- Same design as 16x16 but higher resolution

### 3. icon128.png (128x128 pixels)
- Large icon for Chrome Web Store
- High-quality shield design
- Include "InferShield" text if space allows

## Design Guidelines

**Color Scheme:**
- Primary: #00ff88 (bright green)
- Background: #1a1a1a (dark gray)
- Accent: #ffffff (white)

**Style:**
- Modern, minimal
- Terminal/hacker aesthetic
- Shield or lock symbol
- Optional: Matrix-style elements
