## Permission Justifications (For Chrome Review)

**storage:**
```
Store user settings including API endpoint URL, API key, enabled platforms (ChatGPT/Claude/Gemini/Copilot), and protection mode preferences (Block/Warn). This allows your configuration to persist across browser sessions without requiring re-authentication.
```

**activeTab:**
```
Detect when you're on a supported AI chat platform (chat.openai.com, claude.ai, gemini.google.com, or github.com/copilot) to activate InferShield's real-time scanning. We only access the active tab when you're on a supported site. InferShield does not monitor tabs you're not actively using.
```

**host_permissions (chat.openai.com, claude.ai, gemini.google.com, github.com):**
```
Inject our content script into supported AI chat platforms to scan messages in real-time before you send them. InferShield only works on the platforms you explicitly enable in extension settings. We do not access any other websites or track your browsing activity. Our content script runs only on these four domains to provide AI chat protection.
```
