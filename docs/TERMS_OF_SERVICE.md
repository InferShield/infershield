## 2. Description of Service

InferShield is a browser extension that scans text for personally identifiable information (PII) and sensitive data before you send it to AI chat platforms (ChatGPT, Claude, Gemini, GitHub Copilot). The Service alerts you when sensitive information is detected and allows you to cancel, redact, or proceed with sending your message.
